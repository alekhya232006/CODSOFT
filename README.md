# CODSOFT Python Internship

This repository contains Python projects completed as part of the CodSoft Python Programming Internship.

## Projects

### 1. Calculator

A simple calculator that performs:

* Addition
* Subtraction
* Multiplication
* Division

### 2. Password Generator

Generates secure and random passwords based on user-defined length.

### 3. To-Do List

A command-line application to:

* Add tasks
* View tasks
* Remove tasks

### 4. Rock-Paper-Scissors Game

A Python game where the user plays against the computer.

* Random computer choice
* Winner determination
* Multiple rounds support

### 5. Contact Book

A contact management application that allows users to:

* Add contacts
* View contacts
* Search contacts
* Update contact details
* Delete contacts
